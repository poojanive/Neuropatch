# 🧠 NeuroPatch: A Smart Skin Patch for Brain Monitoring and Response

##  Overview

NeuroPatch is a research-based wearable healthcare system designed to continuously monitor physiological signals and provide real-time responses during critical neurological conditions. The system integrates embedded technology and IoT to enable remote patient monitoring and timely alert generation.

The project aims to provide a portable, cost-effective alternative to conventional hospital-based monitoring systems, improving accessibility and patient safety.

---

##  Problem Statement

Traditional brain monitoring systems are bulky, expensive, and confined to hospital environments, making continuous patient monitoring difficult. Delays in identifying critical neurological conditions can affect timely medical intervention.

---

##  Proposed Solution

NeuroPatch is a compact wearable smart skin patch that uses an ECG sensor and electrode patches to acquire physiological signals. The ESP32 processes the collected data and sends it to the ThingsBoard IoT platform for real-time monitoring. If abnormal conditions are detected, the system generates instant alerts to support timely medical response.

---

##  Working Principle

1. Electrode patches collect physiological signals through the ECG sensor.
2. The ESP32 acquires and processes the sensor data.
3. The processed data is transmitted to the ThingsBoard IoT platform.
4. The data is monitored continuously in real time.
5. If abnormal conditions are detected, an alert is generated for immediate attention.
